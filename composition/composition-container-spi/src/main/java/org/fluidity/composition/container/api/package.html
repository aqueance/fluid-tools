Service provider interfaces to be used by the dependency injection container implementation.

The implementation is offered a few basic services in the form of a {@link org.fluidity.composition.container.api.ContainerServices} object given to the ContainerProvider whenever it is invoked. These basic services include:

  • {@linkplain org.fluidity.composition.container.api.ComponentCache component caching}
  • {@linkplain org.fluidity.composition.container.api.DependencyInjector dependency injection}
  • {@linkplain org.fluidity.composition.container.api.ContextDefinition context definition}
  • {@linkplain org.fluidity.composition.container.api.RestrictedContainer restricted container}
