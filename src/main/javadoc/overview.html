Fluid Tools is a software composition framework. As a framework, it provides a frame to your application and once you have that frame set up, it becomes virtually invisible, thus allowing you to focus on your code rather than its frame.

Fluid Tools also gives you basic and more advanced utilities that Fluid Tools itself makes extensive use of, internally.

Fluid Tools does not conform to JSR 330: Dependency Injection for Java and we don't consider that to be a deficiency in Fluid Tools.

To get started with Fluid Tools, see the Getting Started guide. For a complete documentation, see the User Guide.
