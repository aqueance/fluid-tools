Utilities to assist Fluid Tools in very basic tasks.

Fluid Tools Foundation offers a diverse set of tools to deal with frequent small tasks such as:

  • {@linkplain org.fluidity.foundation.Configuration}: injecting configuration as a dependency.
  • {@linkplain org.fluidity.foundation.Log Logging}: injecting a logger as a dependency.
  • {@linkplain org.fluidity.foundation.Deferred DCL}: Double-Checked Locking with volatile acquire/release semantics;
  • {@linkplain org.fluidity.foundation.ClassLoaders Class loading}: working with class loaders;
  • {@linkplain org.fluidity.foundation.Exceptions Exceptions}: working with checked exceptions;
  • {@linkplain org.fluidity.foundation.Generics Type parameters}: working with generic type parameters;
  • {@linkplain org.fluidity.foundation.Proxies}: working with proxies;
  • {@linkplain org.fluidity.foundation.Streams}: working with streams;
  • {@linkplain org.fluidity.foundation.Archives Archives}: working with JAR archives, including an {@linkplain org.fluidity.foundation.jarjar.Handler URL handler} for JAR archives nested in other JAR archives<;/li>
